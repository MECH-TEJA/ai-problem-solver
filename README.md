# CodeMind: AI-Powered Problem Solver 🤖📚\n\n> Unified Learning Platform for Sciences & Programming\n\n## 🌟 Features\n\n✅ **Multi-Domain Problem Solving**\n- Mathematics, Physics, Chemistry, and Programming (C, C++, Java, Python)\n- Intelligent problem type detection\n\n✅ **AI-Powered Learning**\n- Step-by-step explanations with reasoning\n- Auto-generated practice questions with adaptive difficulty\n- Real-time code debugging with learning feedback\n- Personalized learning paths\n\n✅ **Student Dashboard**\n- Problem input interface\n- Solution display with explanations\n- Progress tracking & analytics\n- Practice history & performance metrics\n\n✅ **Connected Learning**\n- Learn programming through physics/math applications\n- Real-world problem scenarios\n- Cross-domain concept connections\n\n## 🏗️ Project Structure\n\n```\nai-problem-solver/\n├── frontend/                 # React website\n│   ├── public/\n│   ├── src/\n│   │   ├── components/\n│   │   ├── pages/\n│   │   ├── styles/\n│   │   └── App.js\n│   └── package.json\n├── backend/                  # Python AI engine\n│   ├── ai_engine/\n│   │   ├── problem_solver.py\n│   │   ├── code_debugger.py\n│   │   ├── question_generator.py\n│   │   └── progress_tracker.py\n│   ├── app.py\n│   └── requirements.txt\n├── ABSTRACT.md\n├── README.md\n└── SETUP.md\n```\n\n## 🚀 Quick Start\n\n### Prerequisites\n- Node.js & npm (for frontend)\n- Python 3.8+ (for backend)\n- Git\n\n### Installation\n\n1. **Clone Repository**\n   ```bash\n   git clone https://github.com/MECH-TEJA/ai-problem-solver.git\n   cd ai-problem-solver\n   ```\n\n2. **Setup Backend**\n   ```bash\n   cd backend\n   pip install -r requirements.txt\n   python app.py\n   ```\n   Backend runs on: `http://localhost:5000`\n\n3. **Setup Frontend**\n   ```bash\n   cd frontend\n   npm install\n   npm start\n   ```\n   Frontend runs on: `http://localhost:3000`\n\n## 📖 How It Works\n\n### 1. Student Problem Input\n- Student enters a problem (math, physics, chemistry, or code)\n- Platform detects problem type via NLP\n\n### 2. AI Processing\n- Problem Solver: Generates step-by-step solution\n- Code Debugger: Analyzes code and provides fixes\n- Explanation Engine: Creates clear, educational explanations\n\n### 3. Learning Experience\n- Display solution with reasoning\n- Provide related practice questions\n- Track progress and difficulty adaptation\n\n### 4. Progress Tracking\n- Student dashboard shows performance\n- AI recommends personalized learning paths\n- Cross-domain problem suggestions\n\n## 🤖 AI Components\n\n### Problem Solver\n- Handles math, physics, chemistry problems\n- Generates step-by-step solutions\n- Explains underlying concepts\n\n### Code Debugger\n- Analyzes code for errors\n- Provides educational feedback\n- Suggests improvements\n\n### Question Generator\n- Creates practice questions\n- Adapts difficulty based on performance\n- Links across domains\n\n### Progress Tracker\n- Tracks student performance\n- Identifies weak areas\n- Recommends focus areas\n\n## 📊 Tech Stack\n\n**Frontend:**\n- React.js\n- Axios (API calls)\n- Chart.js (Progress visualization)\n- Tailwind CSS (Styling)\n\n**Backend:**\n- Flask (Python)\n- OpenAI API / Hugging Face (AI)\n- SQLite (Progress storage)\n- CORS enabled\n\n## 🎯 Use Cases\n\n1. **Homework Help** - Students get solutions with explanations\n2. **Concept Learning** - Connect programming to real physics/math\n3. **Code Debugging** - Fix errors with learning feedback\n4. **Exam Prep** - Auto-generated practice questions\n5. **Progress Tracking** - Monitor learning journey\n\n## 📝 API Endpoints\n\n```\nPOST   /api/solve              - Solve a problem\nPOST   /api/debug              - Debug code\nPOST   /api/generate-questions - Generate practice questions\nGET    /api/progress/:userId   - Get student progress\nGET    /api/recommendations    - Get personalized recommendations\n```\n\n## 🔐 Environment Variables\n\nCreate `.env` in backend folder:\n```\nOPENAI_API_KEY=your_api_key_here\nFLASK_ENV=development\nDATABASE_URL=sqlite:///progress.db\n```\n\n## 📧 Submission Details\n\n- **Abstract**: See `ABSTRACT.md`\n- **Deadline**: June 29, 2026\n- **Team Members**: GOKUL A, Praveen Kanna K.R, Naresh M, TEJA Reddy, Sujeesh Kumar S, Mallesh Kumar R\n\n## 🎓 Learning Impact\n\nCodeMind transforms education by:\n- Making complex concepts accessible\n- Connecting theory to applications\n- Providing 24/7 personalized tutoring\n- Improving retention through interactive learning\n- Preparing students for integrated STEM careers\n\n## 🤝 Contributing\n\nTeam collaboration welcome!
